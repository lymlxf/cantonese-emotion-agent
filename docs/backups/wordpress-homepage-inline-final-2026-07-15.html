Project Overview

This project studies Cantonese Speech Emotion Recognition by combining audio and textual features for emotion classification and emotion-aware interaction. The work focuses on building a multimodal recognition pipeline that can support Cantonese voice interfaces while clearly separating implemented components from ongoing experiments.
✓Cantonese speech analysis
✓Text-informed emotion modelling
✓Emotion-aware interaction design

How It Works

The system follows a four-stage workflow that connects Cantonese speech input with multimodal emotion recognition and response generation.

1

Cantonese Voice Input

The user provides Cantonese speech through the demonstration interface.
2

Audio and Text Preprocessing

Speech signals and transcribed text are prepared for downstream modelling.
3

Multimodal Emotion Recognition

Audio and textual representations are combined to support emotion classification.
4

Emotion-Aware Response

The interaction layer uses the predicted emotional context to guide responses.

Recognized Emotions

The recognition task is structured around eight emotion categories used for classification and interface feedback.

Angry Disgusted Fearful Happy
Neutral Other Sad Surprised

System Components

The prototype is organized into modular components for data preparation, multimodal encoding, classification, and demonstration.

Data Preprocessing

Prepares Cantonese speech and textual data for model training and evaluation workflows.

Audio Encoder

Extracts speech-based representations that support emotion recognition from acoustic patterns.

Text Encoder

Encodes transcribed linguistic information to complement acoustic emotion cues.

Cross-Modal Attention

Combines audio and text features so the model can use information from both modalities.

Emotion Classifier

Maps multimodal representations to the defined Cantonese speech emotion categories.

Cantonese AI Voice Agent

Provides an interface direction for emotion-aware Cantonese speech interaction.

Application Scenarios

Potential application contexts are being considered as research directions rather than finalized deployments.

Emotion-Aware Cantonese Voice Agent

Supports voice interactions that consider the user's emotional context.

Cantonese Customer Service Emotion Recognition

Explores how emotion recognition could assist service quality analysis.

Cantonese Language Learning Support

Investigates emotion-aware feedback for learners using Cantonese speech.

Interview and Survey Emotion Analysis

Considers structured analysis of emotional signals in spoken responses.

Cantonese Media Content Analysis

Examines possible use in analyzing emotional expression in Cantonese media.
